Challenge: The Curse of Relative Paths

The first major hurdle: linking the CSS file to the HTML. Spent 40 minutes staring at a plain black-and-white page, realizing I had used **`/style.css`** (absolute path from the root) instead of **`../style.css`** (relative path for subfolders). The machine is mercilessly literal.

Breakthrough: Finally Understanding the `alt` Attribute!

That beautiful moment when Leon drilled home that **`alt`** is not just a description, but a **substitute**. It's the text that replaces the image when it fails to load or for screen readers. It clicked that this tiny attribute is the first, simplest layer of true **developer empathy**.
